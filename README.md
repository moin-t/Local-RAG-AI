# Local RAG AI: Interactive Document Q&A Assistant

An offline AI system that allows users to query documents (PDF, TXT, DOCX, CSV) using a local Retrieval-Augmented Generation (RAG) pipeline with Ollama and LangChain.

## Features
- Multi-format document support
- Text chunking & semantic embedding
- Vector database support: Chroma or FAISS
- Interactive Streamlit chat interface
- Customizable prompts for accurate answers
